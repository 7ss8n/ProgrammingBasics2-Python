Завдання 1. Розробити модуль для створення веб-карти. На веб-карті повинна зображатися інформація про місця де знімалися фільми, 
які були зняті того чи іншого року. Користувач вказує для фільмів якого року він бажає побудувати карту і як результат отримує html файл.
Карта повинна мати не менше трьох шарів. За бажанням критерії за якими буде обрано що зображати на карті можуть бути змінені. 
Для отримання координат локацій де знімали фільми можна скористатися будь яким з доступних сервісів за допомогою 
бібліотек geopy, geocoder тощо.

Завдання оформити як окремий проект, розмістити в репозитарії на GitHub і додати посилання на нього як результат виконання завдання на програмування №1.

Файл README.MD проекту повинен містити:

- інформацію про призначення модуля;

- опис структури html файлу тегів розмітки, які були згенеровані;

- короткий висновок інформацію, яку нам надає така карта.
